## Create virtual env

```
python3 -m venv env
```

## Activate virtual env 

```
source env/bin/activate
```

## quit environment

```
deactivate
```

## Install dependencies

```
pip3 install -r requirements.txt
```

## Save dependencies

```
pip freeze > requirements.txt
```
